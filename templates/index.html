A machine reads pieces, not words.

When you type a sentence, the AI never sees whole words. It breaks text into fragments called tokens, which can be a full word, part of a word, or just punctuation. The word "unbelievable" might become three tokens while "hi" is just one.

0 tokens

Every token is a place. Similar meanings sit close.

Each token gets mapped to a point in a space with hundreds of dimensions. Tokens with related meanings end up near each other, so "Paris" and "Tokyo" are neighbors while "banana" is far away. This lets the model treat meaning like geometry.

To pick the next word, the model looks back.

Before choosing what comes next, the model scans everything it has already read and weights each word by how relevant it seems. A word like "chef" carries a lot of influence over what follows. This process is called attention, and it runs on every token at every step.

    The output is a probability over every word.

    The model scores every word in its vocabulary and converts those scores into probabilities. It does not just pick the top word every time. Settings like temperature control how adventurous the final choice is, letting you trade predictability for variety.

    Pick one. Append. Repeat.

    Once a word is chosen, it joins the input and the whole process runs again from the top. The model reads the growing sequence, picks another word, and the cycle continues until it decides to stop. This loop is how a short prompt becomes a full response.
